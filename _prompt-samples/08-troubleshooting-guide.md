# トラブルシューティングガイドプロンプト

## 用途
- よくある問題と解決方法をまとめたトラブルシューティングガイドを作成
- 実際の運用で遭遇する問題の診断と対処法

## プロンプト

```
[Azureサービス/シナリオ]でよく発生する問題とトラブルシューティング方法について、[試験名]試験対策として包括的なガイドを作成して。

# 対象とする問題領域
- パフォーマンス問題
- 接続・ネットワーク問題
- 認証・認可問題
- 設定ミスによる問題
- スケーリング問題
- セキュリティ関連問題

# 各問題について含める要素
- 症状の説明
- 根本原因の特定方法
- 段階的な解決手順
- 予防策とベストプラクティス
- 監視とアラート設定
- 関連するログとメトリクス

# 出力形式
- 問題カテゴリ別の整理
- 診断フローチャート
- 具体的なコマンドと設定例
- よくある間違いと対処法
- 参考リンクとドキュメント
```
